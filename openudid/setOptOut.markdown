## Overview

Allows for disabling of the UDID.


## Syntax

	openudid.setOptOut( optOut )

##### optOut ~^(required)^~
_[Boolean][api.type.Boolean]._ Controls whether UDID tracking is disabled or not.


## Example

``````lua
local openudid = require( "plugin.openudid" )
openudid.setOptOut( true )
``````
