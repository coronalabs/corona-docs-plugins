## Project Settings

To use this plugin, add two entries into the `plugins` table of `build.settings`. When added, the build server will integrate the plugin during the build phase.

``````lua
settings =
{
	plugins =
	{
		["plugin.fuse"] =
		{
			publisherId = "com.coronalabs"
		},
		["plugin.fuse.ads"] =
		{
			publisherId = "com.coronalabs"
		},
	},		
}
``````
