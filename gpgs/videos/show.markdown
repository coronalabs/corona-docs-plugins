## Overview

Shows the video capturing view. Using this, the player can start the capturing process.

## Syntax

	gpgs.videos.show( listener )

##### listener ~^(optional)^~
_[Listener][api.type.Listener]._ Listener function which receives a [show][plugin.gpgs.videos.event.show] event.
