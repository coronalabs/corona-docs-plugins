## Overview

The type of event. Can be one of the following values:

* `"adLoaded"`
* `"adOpened"` (iOS only)
* `"adClosed"`
* `"adClicked"`
* `"adError"`
* `"videoEnded"` (fullscreen and rewarded ads only)
